# 首次使用电脑问题调查表

**调查目的**：了解人们第一次接触电脑时遇到的真实困难，定位"新手盲区"，为编写教程 / 设计引导提供依据。

**填写说明**：
- 调查对象：任何"还记得自己第一次用电脑是什么状态"的人（不限年龄）。
- 题型：单选 = 选一个；多选 = 可多选；开放 = 自由填写。
- 建议样本量 ≥ 30 份才有参考价值；可用「问卷星 / 腾讯文档 / 飞书多维表」直接导入本表发放。

## 二、问题分类速览（用于出报告）

| 硬件与开机 | Q3–Q5 | 是否连"开机 / 基本外设操作"都卡 |

## 三、发放与分析建议

1. **发放渠道**：家族群、社区老年大学、公司新人培训、学校机房都适合；线上用问卷星/腾讯文档，线下可打印 Q1–Q32 纸质版。
2. **交叉分析**：把 Q1/Q2（接触电脑的年代 / 年龄）和后面各维度交叉，能看出"哪个年龄段、哪个年代的人"在哪些题上卡得最厉害。
3. **优先级**：Q32 的"最想获得的帮助"多选结果，直接决定你下一步该先写哪类教程。
4. **开放题**：Q5 / Q14 / Q26 / Q31 的填空里常藏着最真实、最具体的痛点，建议逐条归类（如"乱按电源""拔插头""叫孩子"）做成词云。
