# 写第一份文档

写点东西是电脑最基本的用途。你可以从最简单的工具开始。

## 记事本：零门槛

1. 开始菜单搜「记事本」打开。
2. 直接打字，它就是个纯文字编辑器。
3. `Ctrl + S` 保存，选个文件夹、起个名（如 `购物清单.txt`）。

> 记事本适合随手记、写纯文字。它不认识格式（加粗、颜色都没有）。

## Word：功能更全

需要排版、插图、目录，用 **Word**（Office 套件里）：

- 开头先想好**保存在哪**，`Ctrl + S` 早点按、常按。
- 字号、加粗、标题层级都在顶部工具栏。
- 养成习惯：写到一段就 `Ctrl + S`，防止停电 / 卡死丢稿。

## 保存与打开

- **保存** `Ctrl + S`；**另存为** `Ctrl + Shift + S` 可存一份副本。
- 打开文件：双击它，或在程序里「文件 → 打开」找到它。

> 文档后缀：记事本是 `.txt`，Word 默认是 `.docx`。发给别人前确认对方能打开。
