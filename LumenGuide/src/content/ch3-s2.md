# 开始菜单与任务栏

如果你只记两个地方，就记这两个：**开始菜单**和**任务栏**。

## 开始菜单：所有程序的目录

点屏幕左下角的 Windows 图标（或按键盘上的 `Win` 键），开始菜单就弹出来了。它通常分两栏：

- **左边**：常用程序、最近打开的文件，下面「所有应用」能列出电脑里装的全部软件。
- **右边**：文档、图片、设置、电源等快捷入口。

> 找不到某个软件？直接在点开开始菜单后打字搜索，比翻文件夹快得多。

## 把常用程序“钉”在任务栏

想让某个软件一点就开，不用每次都翻开始菜单：

1. 在开始菜单里找到它，右键 → **“固定到任务栏”**。
2. 之后它就常驻任务栏，点一下就启动。
3. 不想要了，在任务栏上右键它 → **“从任务栏取消固定”**。

## 关机和重启在哪

点开始菜单左下角的**电源按钮**，就能看到「睡眠 / 关机 / 重启」。

- **睡眠**：暂时休息，开盖或动鼠标立刻回来，适合短时间离开。
- **关机**：彻底断电，长期不用时选它。
- **重启**：系统卡了、装完软件要求重启时用。
