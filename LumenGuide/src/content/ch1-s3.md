# 初次使用的基本操作

电脑的操作，归根结底就是**用鼠标和键盘，去控制一个个窗口**。

## 鼠标三件事

- **单击**：选中一个东西。
- **双击**：打开一个东西（文件、程序）。
- **右键**：弹出菜单（后面我们会专门讲右键菜单）。

## 窗口的基本操作

每一个程序都是一个「窗口」，窗口右上角通常有三个按钮：

| 按钮 | 作用 |
| --- | --- |
| 最小化 `_` | 收到底部任务栏，不关闭 |
| 最大化 `▢` | 占满整个屏幕 |
| 关闭 `✕` | 退出程序 |

## 复制与粘贴

这是用得最多的组合：

1. 选中文字或文件；
2. 按 `Ctrl + C` 复制；
3. 到目标位置按 `Ctrl + V` 粘贴。

> 对应地，`Ctrl + X` 是「剪切」（移走），`Ctrl + Z` 是「撤销」。记住这三个，效率翻倍。

学会这些，你已经能完成 80% 的日常操作了。第二卷我们从「手机用户如何过渡到电脑」讲起。
