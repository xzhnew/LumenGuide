# 账户与登录

开机后第一道门就是“登录”。账户决定了这台电脑归谁、设置同步到哪。

## 两种账户

- **微软账户**：用邮箱登录，设置、文件能跨设备同步，应用商店也能用。适合多数人。
- **本地账户**：只存在这台电脑上，不联网同步。更“封闭”，适合只用一台机、在意隐私的人。

> 切换 / 添加：设置 → 账户 → 你的信息 / 家庭和其他用户。

## 登录方式

除了密码，还能设更方便的：

- **PIN**：一组数字，专用于这台电脑，解锁快。
- **指纹 / 人脸**：有对应硬件就能用，最省事。
- 设这些都在 **设置 → 账户 → 登录选项**。

## 安全小贴士

- 密码别用 `123456`、生日这种一眼猜到的。
- 不同网站别用同一个密码。
- 忘了密码？微软账户可在线找回；本地账户要提前准备重置盘。
