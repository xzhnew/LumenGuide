# 新建、重命名与移动

文件多了就得整理。这三件事每天都在用。

## 新建文件夹

1. 在资源管理器空白处**右键** → **新建** → **文件夹**。
2. 输入名字（如「2026 旅行照片」），按回车。
3. 把相关文件拖进去即可。

> 建议一层层分类：比如 `工作 / 项目A / 资料`、`生活 / 照片 / 2026`。别把上百个文件平铺在一个文件夹里。

## 重命名

- 选中文件或文件夹，**慢点两下**名字（不是双击打开！），或按 `F2`。
- 输入新名字，回车确认。
- 改名字**不会**改变文件内容，只改显示的称呼。

## 移动文件

- **拖动**：按住文件拖到另一个文件夹，松手即移动。
- **剪切 + 粘贴**：选中后 `Ctrl + X` 剪切，到目标处 `Ctrl + V` 粘贴。
- **复制 + 粘贴**：`Ctrl + C` 复制，`Ctrl + V` 粘贴——原处还会留一份。

> 分不清移动还是复制？拖动到**同一盘符**内是移动，拖到**另一个盘符**通常是复制。不放心就用 `Ctrl+X / Ctrl+V`。
