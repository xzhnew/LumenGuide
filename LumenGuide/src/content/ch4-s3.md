# 复制、删除与回收站

整理文件绕不开“复制”和“删除”，但删错了怎么办？别慌，有回收站。

## 复制：留一份备份

- 选中文件，`Ctrl + C` 复制，到别处 `Ctrl + V` 粘贴，得到一份一模一样的副本。
- 适合：给朋友发照片前先复制一份、重要文档留底。

## 删除与回收站

- 选中文件按 `Delete`，或右键 → **删除**，文件会进**回收站**，不是真没了。
- 回收站就像桌边的废纸篓：东西还在，随时能捡回来。
- **还原**：打开回收站，右键文件 → **还原**，它就回到原来位置。

> 真要彻底删除（绕过回收站）：选中后按 `Shift + Delete`。这一步要谨慎，删了很难恢复。

## 清空回收站

回收站里的文件仍占硬盘空间。确定都不需要时：

1. 在回收站上**右键** → **清空回收站**。
2. 之后里面的东西才真正消失。

> 重要提醒：U 盘、手机里删掉的文件**通常不经过回收站**，删了就真没了，操作要更小心。
