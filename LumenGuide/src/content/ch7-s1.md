# 学会打字

电脑大部分操作最后都落到“打字”上。先把输入法搞清楚，后面事半功倍。

## 中文用拼音输入法

Windows 自带拼音输入法，开机会自动有。基本用法：

- 直接敲拼音，如 `nihao` → 出现「你好」，按空格或数字选词。
- 不会的字，试着用偏旁拼音或手写面板（输入法右键里有）。
- 中英文切换：一般按 `Shift`；中 / 英标点看输入法状态栏。

> 第一次用如果打不出中文，检查右下角输入法是不是切到了英文。按 `Win + 空格` 能在多个输入法间切换。

## 指法：十指分工

不用背，记住大原则：

- 两个大拇指管空格。
- 其余八指各管一列，手指自然落在键盘中排（A S D F / J K L ;）。
- 眼睛尽量看屏幕，别老盯键盘——练几天就熟了。
