# 安装一步步

下载好安装包（`.exe` 或 `.msi`），双击它就开始安装。关键不是“点下一步”，而是**看清每一步**。

## 标准流程

1. 双击安装包，允许它运行（弹窗点“是”）。
2. 看到**许可协议**，一般勾选“我同意”即可。
3. 到「安装选项」这一步要**慢一点**（见下）。
4. 点「安装 / 开始」，等进度条走完。
5. 最后「完成 / 启动」。

## 最容易踩坑的一步：自定义选项

很多安装程序会默认勾选：

- “安装某某管家”“设为主页”“开机启动”等**捆绑项**。
- 这些大多用很小字、默认打勾。

> 做法：安装时选**“自定义 / 高级”**，把不需要的勾**全部取消**，安装路径也尽量别装到 C 盘根目录乱塞。

## 装完确认

- 在开始菜单 / 任务栏能找到它，说明装好了。
- 如果桌面多出好几个不认识的图标，多半是捆绑软件，按下一节卸载掉。
